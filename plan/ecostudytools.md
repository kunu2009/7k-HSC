### üéØ TOOL 11: Economics Mnemonics Bank

**Purpose:** Memory tricks for remembering economic concepts

**Examples:**
1. **MICRO = SMALL (Price Theory)**
   - M = Marshall (Father)
   - I = Individual units
   - C = Ceteris Paribus
   - R = Relative prices
   - O = One market at a time

2. **MACRO = BIG (Income Theory)**
   - M = Money supply
   - A = Aggregates
   - C = Countrywide
   - R = Real output
   - O = Overall economy
   - (Father: Keynes - "K" for "Kaam bada")

3. **Elasticity Degrees: 0-1-‚àû**
   - 0 = Zero response (Vertical line = life-saving drugs)
   - 1 = Unit elastic (Rectangle = Total Revenue constant)
   - ‚àû = Infinite response (Horizontal line = Perfect competition)

4. **GDP to DI Journey: "GREAT NEWS! NEW PURCHASES DAILY"**
   - G = GDP
   - N = GNP (add NFIA)
   - N = NNP (minus Depreciation)
   - N = NI (at factor cost)
   - P = PI (minus retained + add transfers)
   - D = DI (minus direct taxes)

5. **Index Number Methods: "LAST PAPER FISH"**
   - LA = Laspeyre (base quantities - q0)
   - Pa = Paasche (current quantities - q1)
   - Fi = Fisher (ideal - geometric mean)
   - Sh = Shows (what each measures)

6. **Market Forms: "MONOPOLY is ONE POLY"**
   - Mono = One seller
   - Poly = Many sellers
   - Oligo = Few sellers
